# **Pizza Sales Analysis Using SQL**

## Project Overview

This project presents SQL-based analysis of pizza sales data.
The goal is to derive business insights such as total orders, revenue, and sales patterns using SQL queries.

---

## Project File

- Pizza_sales.pdf → Contains SQL queries along with their outputs and analysis.

---

## Key Analysis Performed

- Performed analysis on total orders and revenue
- Identified top-performing pizzas and popular sizes
- Analyzed order patterns based on time and category
- Evaluated sales trends and customer preferences

##### Detailed SQL queries and results are available in the attached PDF file.

---

## Tools & Technologies

- SQL (Structured Query Language)
- GitHub

---

## Insights

- The total number of orders is high, which shows strong customer demand

- A few pizza types are ordered more than others, showing clear customer preferences

- Large pizzas are the most popular sizes

- Most orders are placed during the afternoon and evening (peak hours)

- Some pizza categories perform better than others in terms of sales

- Daily orders are mostly stable, showing consistent business performance

- Top-selling pizzas contribute a large portion of total revenue

- Overall sales show steady growth over time
